# 프로젝트명: 학생 성적 요인 분석

## 📌 프로젝트 개요
- 학생들의 성적에 영향을 미치는 요인을 분석하는 프로젝트입니다.
- 학생의 학습 참여도, 부모의 학교 만족도 등 다양한 변수가 학생의 성적에 미치는 영향을 분석합니다.

- 데이터 전처리 및 시각화
